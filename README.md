# Embedded System and IoT Projects

## Project Overview
This repository contains a Line Follower Robot implementation with complete hardware design, firmware development, and simulation files.

## Repository Structure
- **CAR_IMG/**: Hardware documentation images
- **LineFollower_ARDUINO/**: Arduino source code for line following algorithm
- **LineFollower_PROTEUS/**: Proteus simulation files
- **Documentation.pdf**: Complete project report

## Technical Specifications
- **Microcontroller**: Arduino Uno (ATmega328P)
- **Sensors**: IR reflective sensors for line detection
- **Motor Driver**: L293D dual H-bridge
- **Control Algorithm**: Proportional line following

## Implementation
The system uses IR sensors to detect a black line on white surface and adjusts motor speeds accordingly to maintain path following.

## How to Use
1. Open Arduino code in Arduino IDE for firmware
2. Open Proteus files for circuit simulation
3. Refer to Documentation.pdf for complete details
